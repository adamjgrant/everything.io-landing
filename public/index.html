About

Bootstrap empowers you with CSS and JavaScript for creating common, minimalistic web applications.

Kickstrap is a standard for extending the functionality of Bootstrap with themes, extra UI components, advanced JavaScript via JSPM, and MVC-pattern UI-binding via AngularJS.

Simply put--everything you need to create a web app of any scale, yet nothing you have to use for beginner web developers.

You may not know what to do with all these technology, so we've provided working example templates using these technologies.
